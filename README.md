## Core Concepts

**Parallel Epistemic Field (PEF)**
Non-temporal persistence of admissible interpretations.

**Admissibility Gating**
Interpretive commitments must pass structural constraints.

**Constraint-Forced Collapse Only**
No probabilistic or heuristic resolution of ambiguity.

**Deterministic Governance Spine**
Every decision is reproducible and auditable.
