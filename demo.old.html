        Operator quick reference
        

          
    
            
  • Surface — This console reflects the proxy instance that serves it. Health, recent, and verify are local to this running surface.
    • 
            
  • Refresh cadence — Health, verify, and recent rows poll about every 8 seconds. Metrics poll every 30 seconds. Use Refresh if you want an immediate pull.
    • 
            
  • Backends — JSONL rows can load entry detail into the side panel. AFL / ledger rows are summarised here, but not expanded into raw line JSON through this pane.
    • 
            
  • Verify semantics — If verification says unanchored slice, widen the window or verify from file start. If the chain is red, treat the result as evidentiary failure, not cosmetic irritation.
    • 
            
  • Session PEF — Session PEF is stateful and query-based. It is not inferred from audit rows. Paste the session id you actually want.
    • 
            
  • Reading rule — Green means the chain is behaving. Yellow means constrained or incomplete. Red means the system crossed into failure or unresolved contradiction.
